# Bugspace Pro

**Bugspace Pro** is an enterprise-grade internal vulnerability management system built as a multi-tenant SaaS platform. It gives security teams a structured way to manage programs, receive vulnerability reports from researchers and employees, triage findings, and track them through to resolution — all with strict role-based access control and organization-level data isolation.

Built with **React + TanStack** on the frontend and **Node.js / Express / TypeScript** on the backend, with **Firebase Firestore** as the target database.

→ See [`documentation.md`](./documentation.md) for the full technical reference including API docs, architecture, data models, RBAC, and the development roadmap.
